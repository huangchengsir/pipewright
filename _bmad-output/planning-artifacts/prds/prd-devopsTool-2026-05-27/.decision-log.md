  - 定位:轻量、自托管、开源的 CI/CD + 部署编排平台;对标 Jenkins 但解决其重/难用/无 AI;整合"CI 工具 + Ansible/Kamal + Portainer"三件套。

  - 风险:单人 + 起步不审代码(SSH 进生产,安全面大);缓解=v1/v2 快速上线后按 PR + 全量安全评审,闸口绑"外部用户接入"时点。
